# 🌐 Portfolio Cá Nhân - Kim Oanh

## 📌 Giới thiệu
Đây là website portfolio cá nhân của Kim Oanh – sinh viên ngành Công nghệ thông tin.  
Website được xây dựng nhằm giới thiệu bản thân, kỹ năng, dự án và định hướng nghề nghiệp.

---

## 🚀 Công nghệ sử dụng
- WordPress
- HTML, CSS
- Plugin:
  - Yoast SEO (tối ưu SEO)
  - WPS Hide Login (bảo mật)
  - Site Kit by Google (Google Analytics)
  - Ad Inserter (hiển thị banner)

---

## 📂 Chức năng chính

### 🏠 Trang chủ (Home)
- Giới thiệu bản thân
- Định hướng nghề nghiệp
- Hình ảnh cá nhân

### 📘 Kỹ năng & Học vấn (Resume)
- Thông tin học tập tại Trường Đại học Tây Nguyên
- Kỹ năng lập trình: C++, Python
- Công cụ: Git, Linux, VSCode

### 💼 Dự án (Projects)
- Danh sách các dự án cá nhân
- Mô tả chi tiết từng dự án
- Hình ảnh minh họa
- Link GitHub

### 📞 Liên hệ (Contact)
- Email
- Mạng xã hội
- Form liên hệ
- Google Maps

---

## 🔐 Bảo mật & SEO
- Đã đổi đường dẫn đăng nhập bằng WPS Hide Login
- Tối ưu SEO bằng Yoast SEO
- Tích hợp Google Analytics để theo dõi người dùng
